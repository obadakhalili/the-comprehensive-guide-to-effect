# The Comprehensive Guide to Effect

A hands-on guide to [Effect](https://effect.website) — what it is, why it exists, and how its runtime
works under the hood. There's a polished article version at **https://effect-guide.netlify.app**.

## What's here

- `tutorial/01-what-is-effect/` — **Part 1**: why Effect exists, using the real `effect` library.
- `tutorial/02-under-the-hood/` — **Part 2**: a small Effect runtime rebuilt from scratch in six steps,
  then used to build a real program.
- `site/` — the static HTML article version of the tutorial.

## Start here

Install once with [Bun](https://bun.sh): `bun install`. Then read the `README.md` in a tutorial folder
and run the files next to it:

```bash
bun tutorial/01-what-is-effect/01-errors-without-effect.ts
```

Every file runs and prints output.
